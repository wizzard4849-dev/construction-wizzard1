# Construction Wizzard – Deploy-Ready Demo

This is a minimal Next.js project you can deploy to Vercel right away.

## What is included
- Landing page at `/`
- Customer demo page at `/demo`
- No environment variables required for the demo build
- No database required for the first deployment

## Local run
```bash
npm install
npm run dev
```
Open `http://localhost:3000`

## Deploy to Vercel
1. Upload this folder to GitHub
2. Import the repo into Vercel
3. Deploy
4. Add your domain `constructionwizzard.com`

## Customer link
After deploy, send customers to:
- `/demo`

Example:
- `https://constructionwizzard.com/demo`
